[color=blue][b][size=12pt][u]Introduction[/u][/size][/b][/color]
This mod allows the users of the Personal Messaging (PM) system to search sent PMs in addition to the core Inbox PMs.

[color=blue][b][size=12pt][u]Admin Settings[/u][/size][/b][/color]
There are no admin settings.  To disable it, you must remove this mod.

[color=blue][b][size=12pt][u]Compatibility Notes[/u][/size][/b][/color]
This mod was tested on SMF 2.0.8, but should work on SMF 2.0 and up.  SMF 1.1 is not and will not be supported, so please don't ask.

This mod will [b]NOT[/b] be compatible with [url=http://custom.simplemachines.org/mods/index.php?mod=3736]Enhanced PM System[/url] versions 2.0 and higher, as that mod will have this mod integrated into it.  It may be used with versions below v2.0.
